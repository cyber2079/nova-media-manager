## 二、代码存放规则

### 2.1 公库 (nova-media-manager)

| 路径 | 内容 | 说明 |
|------|------|------|
| `src/webgl3d/` | 前端全部 3D 代码 | TypeScript/JSX/GLSL |
| `src-tauri/src/commands/webgl3d.rs` | Rust 侧 3D 接口 | 单文件或多模块目录 |
| `scripts/webgl3d/` | 打包/校验脚本 | Python 3 |
| `docs/webgl3d-spec/` | 规范文档 | Markdown |

### 2.2 私库 (nova-proprietary)

| 路径 | 内容 | 说明 |
|------|------|------|
| `themes/webgl3d/prompts/` | AI 生成提示词 | 按主题分目录 |
| `themes/webgl3d/metadata/` | 主题配置（非加密部分） | JSON |

### 2.3 素材目录 (nova-themes-assets, Syncthing)

| 路径 | 内容 | 说明 |
|------|------|------|
| `webgl3d/{theme_id}/models/` | 3D 模型 | glTF 2.0 / GLB |
| `webgl3d/{theme_id}/textures/` | 贴图 | PNG/KTX2/Basis |
| `webgl3d/{theme_id}/animations/` | 动画 | glTF 内嵌或独立文件 |
| `webgl3d/{theme_id}/source/` | 源文件 | Blender/其他工具项目文件 |

## 三、版本控制规则

### 3.1 公库 .gitignore 新增项

```gitignore
# 3D 主题素材 — 不入公库，走 Syncthing 同步
public/themes/
nova-themes-assets/

# 3D 模块打包产物
src/webgl3d/build/
scripts/webgl3d/__pycache__/
scripts/webgl3d/output/*.nv3d
```

### 3.2 私库 .gitignore

```gitignore
# 私钥和敏感配置（已在现有规则中）
# 3D 主题业务密钥（如有新增）
```

### 3.3 Syncthing 同步规则

与现有 `public/themes/` 同步方案一致：
- `D:\nova-themes-assets\webgl3d\` 在台式机和笔记本之间双向同步
- 不加入任何 Git 仓库

## 四、NV3D 打包源目录结构

打包脚本 `scripts/webgl3d/pack_nv3d.py` 的输入源目录结构：

```
{source_dir}/                       ← 打包脚本的输入目录
├── manifest.json                   ← 主题清单（Schema 见 11）
├── i18n/                           ← 多语言文本
│   ├── zh.json
│   ├── en.json
│   ├── ja.json
│   ├── ko.json
│   ├── de.json
│   ├── fr.json
│   └── it.json
├── models/                         ← 3D 模型
│   ├── scene_low.glb               # 低精度预览
│   ├── scene_hd.glb                # 高精度完整模型
│   ├── character.glb
│   └── props/
│       ├── item_001.glb
│       └── ...
├── textures/                       ← 贴图（独立或 KTX2）
│   ├── scene/
│   ├── character/
│   └── props/
├── animations/                     ← 动画
│   ├── idle.glb
│   ├── interaction_01.glb
│   └── ...
├── shaders/                        ← 主题专属着色器（如有）
│   └── (主题特定的 GLSL 变体)
├── audio/                          ← 音效/BGM（如有）
│   ├── bgm.ogg
│   └── sfx/
├── fonts/                          ← 主题专属字体（如有）
└── preview/                        ← 预览图
    ├── thumbnail.webp              # 缩略图 (256×256)
    └── hero.webp                   # 主图 (1920×1080)
```

### 4.1 打包产物

运行打包脚本后输出：

```
scripts/webgl3d/output/
└── {theme_id}_v{version}.nv3d      ← 单一 NV3D 文件
```

### 4.2 目录隔离保障

- 3D 打包源目录完全独立于现有普通主题素材目录
- 脚本的输入和输出均在 `scripts/webgl3d/` 下管理
- 不覆盖、不引用、不修改任何现有主题文件
