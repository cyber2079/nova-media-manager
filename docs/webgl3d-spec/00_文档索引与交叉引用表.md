## 关键术语速查

| 术语 | 定义 | 详见文档 |
|------|------|---------|
| NVTP | 现有普通主题包格式（ZIP + XOR/SHA256，仅用于非 3D 主题） | 17 |
| NV3D | WebGL 3D 主题专属容器格式（Manifest 索引 + 独立分块 + AES-256-GCM + Ed25519 签名） | 08, 17 |
| Lockstep Development | Cyber-Girl 与古风水墨双主题同步开发、互相验证 | 00_总述, 18 |
| 六层纵深防线 | 编译→加载→渲染→线程→数据→运行时 六层隔离 | 00_总述 |
| Error Boundary | React 组件级异常捕获，3D 画布崩溃不影响父组件树 | 12 |
| 静态 2D 降级 | 3D 不可用时回退主题预览图，不做软件渲染 | 05 |
| 扩展风格参数集 | 着色器可配置的视觉参数，通用底层无风格硬编码 | 06 |
| Feature Flag | 全局功能开关 `WEBGL3D_ENABLED`，开发阶段默认 false | 02 |

## 文档维护规则

1. 本文档是**活文档**——任何其他文档的新增、删除、合并，必须同步更新本文档
2. 交叉引用矩阵中的"不覆盖范围"若指向不存在的文档编号，视为本文档过期
3. 每份文档头部须标注版本号和冻结/更新日期
4. P0 文档冻结后，P1-P3 文档基于冻结版本编写；P0 若修改须通知所有下游文档作者
5. 文档中的技术方案不固化框架名、版本号，仅描述接口契约和行为约束
