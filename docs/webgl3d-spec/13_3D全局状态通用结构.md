## 一、状态隔离原则

```
全局 Zustand stores                 3D 模块独立 store
─────────────────                   ──────────────────
gameStore                            threeDStore
settingsStore                        （仅在 webgl3d/state/ 内部可见）
widgetStore
... 3D 模块不读写这些 ...

     主应用不 import threeDStore
```

**关键约束**：
- 3D 模块的状态不写入主应用的任何 store
- 主应用的状态不直接被 3D 模块修改
- 需要数据交互时，通过 bridge/ 中转（如读取设置中的语言偏好）

## 二、threeDStore 结构

### 2.1 状态 Schema

```typescript
interface ThreeDState {
  // ========== 模块级状态 ==========
  module: {
    status: "uninitialized" | "loading" | "active" | "degraded" | "disabled";
    errorInfo: string | null;
    circuitBreakerOpen: boolean;
  };

  // ========== 加载状态 ==========
  loading: {
    phase: "idle" | "manifest" | "low_res" | "hd_streaming" | "complete";
    progress: number;             // 0-100
    currentItem: string | null;   // 当前加载的资源名称
    error: string | null;
  };

  // ========== 场景状态 ==========
  scene: {
    currentSceneId: string | null;
    themeId: string | null;
    themeMeta: ThemeMeta | null;   // 来自 manifest 的 themeName 等
    transition: "none" | "fade_in" | "fade_out" | "loading";
  };

  // ========== 角色状态（可选扩展字段）==========
  characters: Record<string, CharacterState>;

  // ========== 道具状态（可选扩展字段）==========
  props: Record<string, PropState>;

  // ========== 任务状态（可选扩展字段）==========
  quests: Record<string, QuestState>;

  // ========== 性能指标（只读）==========
  performance: {
    fps: number;
    frameTime: number;
    drawCalls: number;
    triangles: number;
    qualityLevel: "high" | "medium" | "low";
  };

  // ========== 交互状态 ==========
  interaction: {
    hoveredObjectId: string | null;
    selectedObjectId: string | null;
    draggingPropId: string | null;
    activeDialogId: string | null;
    isCameraMoving: boolean;
  };

  // ========== 对话状态 ==========
  dialog: {
    currentDialogId: string | null;
    currentLineIndex: number;
    isTyping: boolean;            // 打字机效果进行中
    choices: DialogChoice[];      // 当前对话选项
  };
}
```

### 2.2 子类型定义

```typescript
interface CharacterState {
  visible: boolean;
  currentAnimation: string;
  position: [number, number, number];
  affectionLevel: number;        // 好感度（默认 0，主题自定义范围）
  interactionCount: number;
  lastInteractionAt: string | null;
}

interface PropState {
  placed: boolean;
  position: [number, number, number];
  snappedTo: string | null;      // 吸附到的 snapTarget ID
  inInventory: boolean;
  lastMovedAt: string | null;
}

interface QuestState {
  status: "locked" | "unlocked" | "in_progress" | "completed";
  currentStage: string | null;
  completedStages: string[];
  unlockedAt: string | null;
  completedAt: string | null;
}

interface DialogChoice {
  id: string;
  textKey: string;               // i18n key
  nextDialogId?: string;
  action?: InteractionAction;
}
```

## 三、可选扩展字段设计

### 3.1 极简主题（无角色/道具/任务）

一个"纯观赏"主题只包含场景渲染：

```typescript
// 极简主题的状态快照
{
  module: { status: "active" },
  loading: { phase: "complete", progress: 100 },
  scene: { currentSceneId: "garden", themeId: "zen-garden" },
  characters: {},     // 空对象 — 无角色
  props: {},          // 空对象 — 无道具
  quests: {},         // 空对象 — 无任务
  performance: { ... },
  interaction: { isCameraMoving: false }
}
```

**规则**：所有消费 `characters`/`props`/`quests` 的逻辑在读取前检查字段是否为空，为空时跳过对应分支。

### 3.2 中等主题（有角色无任务）

```typescript
{
  characters: {
    "guide_spirit": {
      visible: true,
      currentAnimation: "idle",
      affectionLevel: 3,
      ...
    }
  },
  props: {
    "incense_burner": { placed: true, ... }
  },
  quests: {}  // 无任务
}
```

### 3.3 完整主题（全模块）

```typescript
{
  characters: { /* 一个或多个角色 */ },
  props: { /* 一个或多个道具 */ },
  quests: { /* 一个或多个任务 */ },
  // 全部字段完整填充
}
```

## 四、Zustand Store 实现

### 4.1 创建方式

```typescript
// src/webgl3d/state/threeDStore.ts
import { create } from "zustand";

export const useThreeDStore = create<ThreeDState>((set, get) => ({
  // 初始状态
  module: { status: "uninitialized", errorInfo: null, circuitBreakerOpen: false },
  loading: { phase: "idle", progress: 0, currentItem: null, error: null },
  scene: { currentSceneId: null, themeId: null, themeMeta: null, transition: "none" },
  characters: {},
  props: {},
  quests: {},
  performance: { fps: 0, frameTime: 0, drawCalls: 0, triangles: 0, qualityLevel: "high" },
  interaction: {
    hoveredObjectId: null, selectedObjectId: null, draggingPropId: null,
    activeDialogId: null, isCameraMoving: false,
  },
  dialog: { currentDialogId: null, currentLineIndex: 0, isTyping: false, choices: [] },

  // Actions（示例）
  setModuleStatus: (status, errorInfo?) =>
    set((s) => ({ module: { ...s.module, status, errorInfo: errorInfo ?? null } })),

  setLoadingProgress: (phase, progress, currentItem?) =>
    set((s) => ({ loading: { ...s.loading, phase, progress, currentItem: currentItem ?? null } })),

  setCurrentScene: (sceneId) =>
    set((s) => ({ scene: { ...s.scene, currentSceneId: sceneId } })),

  updateCharacterState: (charId, patch) =>
    set((s) => ({
      characters: {
        ...s.characters,
        [charId]: { ...s.characters[charId], ...patch },
      },
    })),

  resetAll: () =>
    set({
      module: { status: "uninitialized", errorInfo: null, circuitBreakerOpen: false },
      loading: { phase: "idle", progress: 0, currentItem: null, error: null },
      characters: {},
      props: {},
      quests: {},
      dialog: { currentDialogId: null, currentLineIndex: 0, isTyping: false, choices: [] },
    }),
}));
```

### 4.2 存储隔离

- `threeDStore` 是独立模块，不在主应用的 store 索引中
- 仅在 `src/webgl3d/` 内部 import
- 主应用如需获知 3D 模块的状态（如切换壁纸），通过 bridge 的事件回调而非直接订阅 store

## 五、状态持久化

### 5.1 持久化范围

| 状态字段 | 是否持久化 | 存储位置 | 说明 |
|---------|:---:|---------|------|
| `module.status` | ✅ | `webgl3d_settings` 表 | 重启后恢复开关状态 |
| `scene.currentSceneId` | ✅ | `webgl3d_user_data` 表 | 存档的一部分 |
| `characters` | ✅ | 存档 | 角色状态 |
| `props` | ✅ | 存档 | 道具位置 |
| `quests` | ✅ | 存档 | 任务进度 |
| `performance` | ❌ | — | 仅运行时 |
| `loading` | ❌ | — | 仅运行时 |
| `interaction` | ❌ | — | 仅运行时 |
| `dialog` | ❌ | — | 仅运行时 |

### 5.2 存档保存/加载

```typescript
// 保存
async function saveToSlot(themeId: string, slot: number) {
  const state = useThreeDStore.getState();
  const saveData = {
    version: 1,
    themeId,
    updatedAt: new Date().toISOString(),
    scene: { currentSceneId: state.scene.currentSceneId },
    characters: state.characters,
    props: state.props,
    quests: state.quests,
  };
  await invoke("webgl3d_save_data", { themeId, slot, data: JSON.stringify(saveData) });
}

// 加载
async function loadFromSlot(themeId: string, slot: number) {
  const result = await invoke("webgl3d_load_data", { themeId, slot });
  if (result.success && result.data) {
    const saveData = JSON.parse(result.data);
    useThreeDStore.setState({
      scene: { ...useThreeDStore.getState().scene, currentSceneId: saveData.scene.currentSceneId },
      characters: saveData.characters || {},
      props: saveData.props || {},
      quests: saveData.quests || {},
    });
  }
}
```
