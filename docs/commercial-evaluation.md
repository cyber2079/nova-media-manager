## 一、市场分析

### 1.1 目标市场容量

| 细分市场 | 全球用户量 | 中国用户量 | 年增长率 |

| 桌面美化工具 | 2.5 亿 | 5000 万 | 8% |

Nova 面向的是「桌面美化 + 二次元 + 媒体管理」三圈交集用户。Wallpaper Engine 全球 3000 万+ 销量证明了这个交叉市场足够大。

Nova 在右上象限独占 — 兼具媒体管理能力和二次元桌面美化，这是明确的蓝海定位。

| 方案 | 价格 | 包含 | 目标转化率 |

| **月度订阅** | ¥19/月 | + 全部主题 + 自动更新 + 优先支持 | 15% |

## 三、支付与收款方案

### 3.1 推荐支付架构

```
用户 → 支付页面 → 支付网关 → 回调通知 → 激活码生成 → 用户激活
                         │
                    多种支付方式:
                    ├── 支付宝 (当面付/APP支付)
                    ├── 微信支付 (JSAPI)
                    ├── Stripe (国际信用卡)
                    └── 爱发电 (手动, 现有)
```

### 3.2 支付方案对比

| 方案 | 费率 | 接入难度 | 自动化 | 适用场景 |
|------|------|------|------|------|
| **爱发电** (现有) | 0% (平台不收) | ✅ 已接入 | ❌ 手动 | 粉丝赞助 |
| **支付宝当面付** | 0.6% | ⭐⭐ | ✅ 自动 | 国内个人收款 |
| **微信 JSAPI** | 0.6% | ⭐⭐⭐ (需企业资质) | ✅ 自动 | 国内 |
| **Stripe** | 2.9%+$0.30 | ⭐⭐ | ✅ 自动 | 国际 |
| **Lemon Squeezy** | 5%+$0.50 | ⭐ | ✅ 自动 | 全球 MoR |
| **个体户执照** | 免费 | ⭐⭐⭐⭐ (跑政府) | N/A | 合规基础 |

### 3.3 推荐支付方案

**第一阶段 (Beta)**: 爱发电 (手动) + 支付宝转账 (手动)。零成本，验证付费意愿。

**第二阶段 (正式)**: 
- 国内: 支付宝当面付 (个体户执照即可)
- 国际: Lemon Squeezy (MoR 模式，无需公司)
- 自动回调 → 激活码自动发放

**第三阶段 (规模)**: 
- 国内: 微信+支付宝 (需企业资质)
- 国际: Stripe
- 自建支付页面 + 激活码管理后台

## 四、用户获取成本 (CAC) 与生命周期价值 (LTV)

### 4.1 CAC 估算

| 渠道 | 月预算 | 预期注册 | CAC |
|------|------|------|------|
| B站内容营销 | ¥0 (自制) | 200-500 | ¥0 |
| GitHub 自然流量 | ¥0 | 100-300 | ¥0 |
| 知乎/贴吧推广 | ¥500 | 50-150 | ¥3-10 |
| B站 KOL 合作 | ¥3,000 | 500-2,000 | ¥1.5-6 |
| 抖音信息流 | ¥5,000 | 1,000-3,000 | ¥1.7-5 |
| 搜索引擎 SEO | ¥0 (自优化) | 100-200 | ¥0 |

**综合 CAC**: 初期 ¥0-3 (有机增长为主)，规模期 ¥2-8 (含付费投放)

### 4.2 LTV 估算

| 用户类型 | 占比 | 生命周期 | 年均消费 | LTV |
|------|------|------|------|------|
| 免费用户 | 88% | 2 年 | ¥0 | ¥0 |
| 月度订阅 | 6% | 8 个月 | ¥228 | ¥152 |
| 年度订阅 | 4% | 2 年 | ¥298 | ¥596 |
| 永久买断 | 2% | 终身 | ¥299 | ¥299 |
| **加权平均 LTV** | | | | **¥43.70** |

### 4.3 LTV/CAC 分析

LTV/CAC = ¥43.70 / ¥4 ≈ **10.9x**

> 一般认为 LTV/CAC > 3x 即为健康。10.9x 表明有机获客效率极高，有空间加大付费投放。
