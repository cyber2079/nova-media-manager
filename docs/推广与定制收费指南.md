# 冰系媒体管家 — B站推广 & 个人定制收费完整指南

---

## 一、产品定位

- **类型**：本地桌面软件（Windows），基于 Tauri v2 + React + Rust 开发
- **核心功能**：电影管理 / 图片浏览 / 音乐播放+歌单 / 游戏启动器 / PoE 冰系御姐主题 / 打字机文案 / 秘书视频表情
- **收费模式**：**基础免费 + 个人定制收费**

---

## 二、B站推广策略

### 2.1 核心原则

B站只做**内容展示和引流**，不在平台内完成交易。

| 能做 | 不能做 |
|---|---|
| 展示软件功能、录屏演示 | 视频放二维码、微信、QQ号 |
| 简介放 GitHub 地址（开源链接） | 引导"私信领安装包""加微信购买" |
| 讲解开发思路、技术架构 | 视频里说价格、套餐、打折 |
| 评论区正常回复技术问题 | 评论/弹幕/私信发外链 |
| 标题带"自研""Tauri全栈"标签 | 夸大宣传（"全网最强""100%完美"） |

### 2.2 视频内容方向

1. **功能演示型**："我用 Rust 写了一个冰系御姐桌面秘书"
2. **技术分享型**："Tauri v2 桌面应用开发踩坑全记录"
3. **视觉展示型**："PoE 寒冰主题桌面——打字机文案+背景MP4循环"
4. **实用场景型**："本地媒体管理工具对比——为什么不直接用文件夹"

### 2.3 引流路径

```
B站视频（展示功能+Tauri技术标签）
    ↓ 简介放 GitHub 地址
GitHub 仓库 README
    ↓ 功能介绍 + 截图 + Release下载
免费版 Release（.msi / .exe 安装包）
    ↓ README 底部留邮箱
定制需求 → 邮件沟通 → 私域成交
```

### 2.4 坚决不碰的红线

- ❌ 爬虫、第三方客户端、破解工具
- ❌ 视频/封面/评论放外链、网盘、二维码、微信
- ❌ "最强""永久免费""万能"等绝对化广告词
- ❌ 抓取 B站/其他平台数据接口
- ❌ 商业付费软件不报备花火，私自发导流视频

## 四、法律与合规

### 4.1 你不需要做的

| 事项 | 说明 |
|---|---|
| 软件著作权登记 | 桌面个人工具，不联网不收费，不需强制登记 |
| ICP 备案 | 不建网站、不联网收集数据，不需要 |
| 工信部 APP 备案 | 仅适用于手机 APP，桌面软件不适用 |
| 隐私政策 | 建议写一句"本应用不收集任何个人数据"放在 README 里 |
| 花火商单 | 只有投付费商业广告才需要，免费工具不需要 |

### 4.2 你需要注意的

| 风险 | 应对 |
|---|---|
| 客户不付款 | **先付后做**，不赊账 |
| 付款后不满意反复改 | 明确"一次免费修改，超出加钱" |
| 买家二次分发你的定制版 | 安装包可加简单授权码（防普通用户） |
| 税务 | 个人小额不用管，**月流水超3万**考虑个体户执照 |
| 版权素材 | 视频用的背景图、字体确保免费可商用，或自己制作 |
| 开源协议 | 你用了 Tauri/MIT 协议的库，需在项目里标注原作者（README 里列一下就行） |

## 五、FAQ

**Q: 不开源能放 GitHub 吗？**
A: 可以。仓库设为 Private，源代码不公开。Release 页面的安装包是公开可下载的。

**Q: B站放 GitHub 链接会违规吗？**
A: GitHub 属于开源代码托管平台，B站允许简介放开源项目链接。你的项目属于个人工具，完全合规。

**Q: 视频播放量很低怎么办？**
A: 持续发，不要只发一个。技术向+展示向交替发，蹭 Rust、Tauri、桌面开发等长尾关键词搜索流量。

**Q: 能不能接定制但不走邮件，直接微信？**
A: 可以。GitHub README 放微信号也行。但不建议在 B站视频里提微信。

**Q: 客户要求的功能我做不到怎么办？**
A: 坦诚拒绝，或者转介绍给其他开发者（收取介绍费）。不要硬做超出能力的功能。
