宣传片双语提词板
1. 冷启动封面（0:00 ~ 0:05）
画面	Logo 居中淡入
一个本地媒体管理桌面应用。

A local media management desktop app.

2. 首页总览（0:05 ~ 0:20）
操作	冷启动进入首页 → 鼠标缓慢扫过导航栏 [电影/图片/音乐/游戏] → 视线落在统计卡片上
电影、图片、音乐、游戏，四大库统一管理。首页一眼看到各库数量和最近添加。

Movies, images, music, games — four libraries, one dashboard. At a glance: counts, recent items, popular tags.

3. 电影库（0:20 ~ 0:50）
操作	点击"电影" → 滚动展示网格 → 在搜索框输入关键词 → 点击标签筛选 → 点击一部电影，内嵌播放器展开 → 关闭
电影库自动刮削封面与元数据。搜索、标签筛选、批量操作，都在这里。内嵌播放器直接预览，不用跳转。

Auto-scraped covers and metadata. Search, tag filter, batch operations — all in one place. Built-in player lets you preview without leaving.

4. 图片库（0:50 ~ 1:10）
操作	切到"图片" → 滚动浏览 → 点击一张进入查看器 → 展示滚轮缩放 → 关闭
图片库同样支持标签管理，快速分类。点击放大，滚轮缩放，流畅浏览。

Image library with tag management. Click to zoom, scroll to browse. Fast and fluid.

5. 音乐库 + 后台播放（1:10 ~ 1:50）
操作	切到"音乐" → 点击一首播放 → 鼠标指向歌词区滚动 → 切换可视化器模式 → 点击导航切到电影页 → 底部迷你播放器出现 → 操作暂停/下一首/音量 → 点最大化回到音乐页
音乐播放集成歌词同步和可视化器。切走页面自动进入后台模式——底部迷你播放器随时控制。你不用停下来，音乐不会停。

Synced lyrics and audio visualizer. Leave the page, and it switches to background mode — a mini player stays at the bottom. You keep working, the music keeps playing.

6. 游戏库（1:50 ~ 2:05）
操作	切到"游戏" → 鼠标划过游戏列表 → 点击一个游戏启动 → 展示启动画面 → 切回
游戏库关联本地可执行文件，一键启动。统一入口，告别满桌面的快捷方式。

Link local executables, launch with one click. One unified launcher — no more desktop shortcut clutter.

7. 主题切换（2:05 ~ 2:45）
操作	打开设置齿轮 → 点击"外观"标签 → 逐一切换主题，每切一个停顿 2~3 秒展示首页变化（默认 → PoE → 赛博 → 黑白，选 4 个代表性的即可）→ 关闭设置
内置多套主题——从最终幻想的魔晄都市，到赛博朋克的霓虹街区。每套主题独有配色、字体、角色图标，甚至首页剧情和语音。切换实时生效。

Multiple built-in themes — from Final Fantasy's Mako City to a cyberpunk neon district. Each with its own colors, fonts, character icons, even story dialogue and voice lines. Switch happens instantly.

8. 桌面小组件（2:45 ~ 3:05）
操作	切到设置 → "小组件"标签 → 逐个开启：时钟 → 日历 → 系统监控 → 倒计时 → 鼠标指向首页上出现的各个小部件
时钟、日历、系统监控、倒计时——可拖动、可配置。按页面或全局显示，放在你需要的地方。

Clock, calendar, system monitor, countdown — draggable, configurable. Show per page or globally, right where you need them.

9. 倒计时 + 语音播报（3:05 ~ 3:20）
操作	双击倒计时小组件 → 设定 10 秒 → 等待归零 → 全屏提醒弹出 + 语音播放 → 点击"我知道了"关闭
倒计时到点触发全屏提醒。不同主题匹配不同角色语音——沉浸式的提醒体验。

Countdown triggers a full-screen alert. Each theme comes with its own character voice — an immersive reminder experience.

10. 快速启动栏 + 收尾（3:20 ~ 3:30）
操作	鼠标滑到底部 → 快速启动栏图标高亮 → 点击一个程序 → 切回首页 → Logo 再次淡入
底部快捷栏常驻常用程序。一切，尽在桌面。

Quick-launch bar always at hand. Everything, right on your desktop.

黑屏 + 下载/官网信息（3:30 结束）
